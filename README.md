# SmartProfile Presentation

A stunning, interactive presentation web application for Real Advancement's SmartProfile service.

## Tech Stack

- React 19
- TypeScript
- Tailwind CSS v4
- Framer Motion
- React Router v7

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Start the development server:
    ```bash
    npm run dev
    ```

3.  Open your browser to `http://localhost:5173` (or the port shown in the terminal).

## Project Structure

- `app/routes/home.tsx`: The main presentation page assembling all sections.
- `app/components/`: Contains all the section components.
- `app/components/ui/`: Reusable UI components (Button, Section, etc.).
- `app/app.css`: Global styles and Tailwind theme configuration (Gold/Black luxury theme).
